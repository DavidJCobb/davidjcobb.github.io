Major works

  • Cobb Positioner (2016)

    An in-game map editor for The Elder Scrolls V: Skyrim.

  • NorthernUI (2018)

    A UI and controls overhaul for The Elder Scrolls IV: Oblivion, relying on handmade graphics and injected C++ code.

  • ItemTrig (2019)

    An inventory management tool for Elder Scrolls Online, written in Lua.

  • ReachVariantTool (2020)

    A script editor for Halo: Reach multiplayer, written in C++ with the Qt UI framework.

Minor works

  • Cobb Bug Fixes (2019)

    An assortment of bug fixes for The Elder Scrolls V: Skyrim's engine, applied via code injection.

  • Cobb Encumbrance (2015)

    A mod for The Elder Scrolls V: Skyrim which uses some unusual tricks to alter how inventory management works.

  • ESO Death and Resurrection (2018)

    A mod for The Elder Scrolls V: Skyrim which implements death and respawn mechanics from Elder Scrolls Online.

  • Skyrim Outfit System (2019)

    A mod for The Elder Scrolls V: Skyrim which uses code injection to alter outfit customization.
